Aide

URL Méthode Résumé
/cv24/delete DELETE Suppression du CV dont l’identifiant est <id>. Retourne un flux XML avec l'ID et le statut de l'opération.
/ GET Affiche la page d'accueil du projet avec les informations sur le nom, version, membres de l'équipe et le logo.
/cv24/insert POST Insertion d'un nouveau CV. Consomme un XML et retourne un XML avec l'ID et le statut de l'opération.
/cv24/xml GET Récupère un CV au format XML en fonction de l'ID.
/cv24/html GET Affiche un CV au format HTML en fonction de l'ID.
/cv24/resume/xml GET Récupère un résumé de tous les CV au format XML.
/cv24/resume GET Affiche un résumé de tous les CV au format HTML.
/cv24/validate-xml POST Valide un XML de CV par rapport au schéma XSD. Retourne un message indiquant si le XML est valide ou non.
